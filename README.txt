# Soundboard Project

This project is a soundboard application that allows users to play various sound effects at the press of a button.

## Features
- Play multiple sound effects.
- User-friendly interface.
- Customizable soundboard layout.

## Requirements
- Python 3.x
- Required libraries: pygame

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/yourusername/soundboard.git
    ```
2. Navigate to the project directory:
    ```
    cd soundboard
    ```
3. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
